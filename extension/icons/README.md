# Иконки расширения

Создайте следующие иконки в формате PNG:

- `icon16.png` - 16x16 пикселей (для панели инструментов)
- `icon48.png` - 48x48 пикселей (для управления расширениями)
- `icon128.png` - 128x128 пикселей (для Chrome Web Store)

Можно использовать любой графический редактор или онлайн-генераторы иконок.

Временное решение: можно использовать простые цветные квадраты или заменить на любые подходящие иконки.
